## Project setup

### Database setup
- Install PostgreSQL
- Open pgAdmin
- Make database user "wad" with password "wad24" (no extra rights needed)
- Make a database named "WAD24_Team-B" with owner "wad"

### Back-end setup
Go into Back-end folder and run:
```
npm install
node server
```

### Front-end setup
Go into Front-end folder and run:
```
npm install
npm run serve
```
